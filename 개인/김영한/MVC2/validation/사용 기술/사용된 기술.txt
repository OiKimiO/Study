1. spring-boot-starter-validation
  - 특징 
    * Starter for using Java Bean Validation with Hibernate Validator
      # 참고 : https://docs.spring.io/spring-boot/docs/current/reference/htmlsingle/
    * 자바 빈 validation과 하이버네이트 validator로 함께 사용됨
    * javax.validation.constraints 등과 같은 유효성 검증 애노테이션을 사용할 수 있도록
      스프링 애플리케이션 컨텍스트에 LocalValidatorFactoryBean을 등록하고, 이를 이용해
      객체의 유효성을 검증
  - 사용법
    * LocalValidatorFactoryBean
      $ javax.validation.ValidatorFactory 인터페이스로 구현된 클래스
      $ 클래스는 스프링 애플리케이션 컨텍스트에 등록됨

2. javax.validation(Interface)
  - 자바 빈 유효성 검증(JSR-303)을 위한 스펙
  - 사용법
    * NotBlank 
      # package : javax.validation.constraints
        $ The annotated element must not be null and must contain at least one non-whitespace character. Accepts CharSequence.
        $ null이 아니고, 문자를 하나 이상 포함해야 함. CharSequence를 사용
    * NotNull
      # package : javax.validation.constraints
    * Max
      # package : javax.validation.constraints
    * Min
      # package : javax.validation.constraints

3. Hibernate Validator(class)
  - javax.validation 스펙을 토대로 검증 기능을 구현한 구현체
    * 하지만 모든 validation 기능이 javax.validation 기능을 준수하는 것은 아님
  - 사용법
    * Range
      # package : org.hibernate.validator.constraints
